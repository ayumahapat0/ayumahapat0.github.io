Author Detection

Along with my teammates, Leo Xu and Ivan Micanovic, we applied different data augmentation methods to a known model called "GhostBusters", a model used to detect AI-Generated text. We applied these data augmentation methods to see if we could improve the model's ability to detect AI-Generated text, specifically in school environments where there is large ethical Concerns using LLMs, such as ChatGPT, to write essays.

Github link to Final Report: SonicStudy
